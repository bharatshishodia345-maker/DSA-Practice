# Time Complexity

## Overview

This folder contains basic examples demonstrating different time complexities used in Data Structures and Algorithms.

---

## Programs

### 1. sum_and_product.py

Calculates the sum and product of a list.

Time Complexity

O(n)

---

### 2. integer_to_string.py

Converts an integer into a string without using Python's built-in conversion.

Time Complexity

O(log n)

---

### 3. nested_loops.py

Demonstrates nested iteration over a list.

Time Complexity

O(n²)

---

### 4. logarithmic_loop.cpp

Example of nested loops where the inner loop doubles each iteration.

Time Complexity

O(n log n)

---

## Concepts Covered

- Big O Notation
- O(1)
- O(log n)
- O(n)
- O(n²)
- O(n log n)
