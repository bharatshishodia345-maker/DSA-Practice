# Dynamic Array Implementation in Python

## 📖 Overview

This folder contains a step-by-step implementation of a Dynamic Array using Python and the `ctypes` module.

The goal is to understand how Python's built-in list works internally by implementing similar functionality from scratch.

---

## 📂 Programs Included

### 1. Create Dynamic Array

- Create a custom dynamic array class.
- Allocate memory using `ctypes`.

### 2. Dynamic Array Length

- Implement the `__len__()` method.
- Return the current number of elements stored.

### 3. Append Operation

- Implement the `append()` method.
- Dynamically resize the array when it becomes full.

# Dynamic Array Implementation in Python

## 📖 Overview

This folder contains a step-by-step implementation of a Dynamic Array from scratch using Python and the `ctypes` module.

The project demonstrates how Python's built-in list works internally by gradually implementing its core functionalities.

---

## 📂 Programs Included

### 01. Create Dynamic Array
- Allocate memory using `ctypes`

### 02. Length Function
- Implement `__len__()`

### 03. Append Operation
- Add new elements
- Automatic resizing

### 04. Print Dynamic Array
- Implement `__str__()`

### 05. Indexing
- Implement `__getitem__()`

### 06. Pop Operation
- Remove the last element

### 07. Clear Operation
- Remove all elements

### 08. Find Operation
- Search an element and return its index

### 09. Complete Dynamic Array
- Combines all implemented features into one class
