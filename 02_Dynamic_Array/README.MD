# Dynamic Array Implementation in Python

## 📖 Overview

This folder contains a step-by-step implementation of a Dynamic Array using Python and the `ctypes` module.

The goal is to understand how Python's built-in list works internally by implementing similar functionality from scratch.

---

## 📂 Programs Included

### 1. Create Dynamic Array

- Create a custom dynamic array class.
- Allocate memory using `ctypes`.

### 2. Dynamic Array Length

- Implement the `__len__()` method.
- Return the current number of elements stored.

### 3. Append Operation

- Implement the `append()` method.
- Dynamically resize the array when it becomes full.
