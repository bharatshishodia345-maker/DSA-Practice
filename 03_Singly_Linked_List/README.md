# Singly Linked List Basics in Python

## 📖 Overview

This module demonstrates the fundamental implementation of a Singly Linked List from scratch using Python classes.

Each program introduces a new Linked List operation step by step to understand how nodes are created, connected, and manipulated.

---

## 📂 Programs Included

### 1. Create Node
- Create a single node
- Understand node structure
- Data and next pointer

### 2. Insert at Head
- Insert a node at the beginning of the linked list
- Update head pointer
- Increase list size

### 3. Traverse Linked List
- Traverse all nodes
- Display linked list in readable format
- Override `__str__()` method

### 4. Append Node
- Insert a node at the end of the linked list
- Traverse to the last node
- Update next pointer

### 5. Insert After Node
- Search a node by value
- Insert a new node after a given node
- Handle value not found case

---

## 📚 Concepts Covered

- Node Creation
- Head Pointer
- Traversal
- Append Operation
- Insert After
- Object-Oriented Programming
- References and Pointers
